# Icon Generation Prompts - Duolingo Style Flat Illustration

## Style Guide
- **Flat & Minimal**: No outlines or very soft edges, clean solid shapes
- **Bright, Saturated Colors**: Bold greens, yellows, blues, purples, reds
- **Rounded Geometry**: Soft corners, organic friendly shapes
- **Cartoonish/Friendly Tone**: Whimsical, not rigid or corporate
- **Low Detail, High Recognition**: Simplified to most recognizable form
- **Consistent Visual Language**: Same style, similar proportions
- **Soft UI Integration**: Rounded rectangle containers with soft glow borders

## Usage Notes
- Generate each icon as a standalone SVG or PNG
- Use dark background (#1A1A1A or similar) for contrast
- Apply subtle soft glow/shadow for selected states
- Maintain consistent stroke weight: 0 (all flat shapes)
- Export at consistent icon sizes (e.g., 64x64, 128x128)
